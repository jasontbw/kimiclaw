# 六六天团使用指南

## 快速启动子代理

### 命令格式

```
sessions_spawn 任务描述 --label="六六天团-[花名]-任务名"
```

### 各成员调用示例

**炳总（运营执行）**
```json
{
  "task": "[炳总-运营执行] 设计一个线上活动的完整执行流程，包括筹备、执行、复盘三个阶段。目标：3天内完成500人参与的行业分享会。",
  "label": "六六天团-炳总-活动执行方案"
}
```

## 协作场景示例

### 场景：举办一场行业大会

```
1. 六六接收任务：举办行业大会
2. 拆解并发包：
   - 炳总：制定执行计划和SOP
   - 海总：设计主视觉和物料规范
   - 叶总：策划传播内容和节奏
   - 修总：分析目标受众和报名转化漏斗
   - 万总：维护重点客户邀请名单
3. 收拢结果，汇总成完整方案
4. 田总拍板确认
5. 炳总接管执行
```

### 场景：新产品上线

```
1. 六六接收任务：新产品上线
2. 拆解并发包：
   - 修总：竞品数据分析和定价建议
   - 叶总：上线传播方案（预热/爆发/持续）
   - 海总：产品视觉包装和页面设计
   - 万总：重点客户提前试用和反馈收集
3. 收拢结果，形成上线方案
4. 田总确认资源投入
5. 炳总制定详细执行计划
```

## 注意事项

1. **任务描述要清晰** - 包括背景、目标、约束条件
2. **指定输出格式** - 明确需要什么形式的产出
3. **设置合理时限** - 子代理默认超时 10 分钟
4. **并行优于串行** - 多个独立任务同时发起

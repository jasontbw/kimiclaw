**原则**: 各管一摊，结果向我汇报

## 协作流程

```
Jason(田总) → 模糊意图
     ↓
    六六（拆解任务）
     ↓
┌────┼────┬────┬────┐
↓    ↓    ↓    ↓    ↓
炳总  修总  叶总  万总  海总
(执行)(数据)(内容)(客户)(视觉)
     └────┼────┘
          ↓
        六六（收拢结果）
          ↓
      田总（确认/拍板）
```

## 调用方式

通过 `sessions_spawn` 动态启动子代理：

```json
{
  "agentId": "main",
  "task": "[角色定位] 具体任务描述",
  "label": "六六天团-炳总-任务名"
}
```

各成员 Skill 文件位于 `/root/.openclaw/skills/liutiantuan/`

# 六六天团职业发展阶梯体系

> **定位**：顶级智慧专家联盟 | **目标**：30-80人外贸公司转型专家

---

## 一、职级体系架构（3+1层级）

```
┌─────────────────────────────────────────────────────────────┐
│                     职级金字塔                              │
├─────────────────────────────────────────────────────────────┤
│  Level 4  ┃  首席合伙人 (Managing Partner)                  │
│           ┃  Miles/六六专属席位 + 极少数破格晋升              │
├───────────┼─────────────────────────────────────────────────┤
│  Level 3  ┃  高级合伙人 (Senior Partner)                     │
│           ┃  某一领域的绝对权威，可独立带队交付大型项目        │
├───────────┼─────────────────────────────────────────────────┤
│  Level 2  ┃  合伙人 (Partner)                               │
│           ┃  独当一面的专家，能主导中型项目交付               │
├───────────┼─────────────────────────────────────────────────┤
│  Level 1  ┃  高级顾问 (Senior Consultant)                   │
│           ┃  专业扎实，能独立完成模块级工作                   │
└───────────┴─────────────────────────────────────────────────┘
```

### 层级定位说明

| 职级 | 英文名 | 核心定位 | 类比参考 |
|------|--------|----------|----------|
| 首席合伙人 | Managing Partner | 战略制定者+组织灵魂 | 麦肯锡高级合伙人/BCG合伙人 |
| 高级合伙人 | Senior Partner | 领域权威+大项目Owner | 四大合伙人/贝恩项目经理 |
| 合伙人 | Partner | 独当一面+中型项目Owner | 咨询公司项目经理 |
| 高级顾问 | Senior Consultant | 专业执行+模块负责人 | 咨询公司高级顾问 |

## 二、晋升标准（按角色分类）

### 2.1 通用晋升标准（所有角色适用）

| 维度 | 高级顾问→合伙人 | 合伙人→高级合伙人 |
|------|-----------------|-------------------|
| **年限要求** | 2-3年 | 3-5年 |
| **项目经验** | 完整参与8+个项目 | 主导交付15+个项目 |
| **客户口碑** | 客户满意度≥4.5/5 | 老客户复购率≥60% |
| **团队贡献** | 培养1名新人/输出方法论 | 建立团队SOP/培养2名合伙人级 |

## 四、横向发展路径

### 4.1 横向转岗机制

```
                    ┌─────────────┐
                    │   高级顾问   │
                    │  (入口层级)  │
                    └──────┬──────┘
                           │
           ┌───────────────┼───────────────┐
           │               │               │
           ▼               ▼               ▼
    ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
    │   专业纵深   │ │   横向拓展   │ │   综合管理   │
    │  (单一专家)  │ │  (T型专家)   │ │  (项目总监)  │
    └─────────────┘ └─────────────┘ └─────────────┘
```

### 4.2 转岗条件与路径

| 转岗类型 | 条件要求 | 路径示例 | 支持政策 |
|----------|----------|----------|----------|
| **相邻模块转岗** | 当前职级满1年+目标模块项目2+个 | 运营总→财务总（都涉及ERP） | 3个月带薪学习期+导师制 |
| **跨域转岗** | 当前职级满2年+目标模块项目4+个 | 社媒总→市场总 | 6个月过渡期+项目补贴 |
| **综合转岗** | 3个模块以上经验+项目管理经验 | 任一专家→项目总监 | 直接晋升合伙人级 |

### 4.3 转岗审批流程

```
个人申请 → 原领域导师评估 → 目标领域合伙人面试 → 
3个月试岗期 → 双向评估 → 正式转岗 or 返回原岗
```

### 4.4 T型人才发展模型

鼓励专家在深耕本专业的同时，横向拓展1-2个相邻领域：

| 本专业 | 推荐横向领域 | 复合价值 |
|--------|--------------|----------|
| 运营总 | 财务总 | 业财一体化专家 |
| 财务总 | 运营总/人事总 | 人效财务专家 |
| 市场总 | 社媒总 | 数字化营销专家 |
| 社媒总 | 市场总/销售总 | 增长黑客专家 |
| 销售总 | 市场总 | 商业化专家 |
| 人事总 | 运营总 | 组织效能专家 |

## 五、降级/退出机制

### 5.1 降级触发条件

| 触发类型 | 具体标准 | 降级处理 |
|----------|----------|----------|
| **业绩不达标** | 连续2个季度项目交付满意度<3.5 | 降一级，6个月观察期 |
| **客户投诉** | 年度客户投诉≥2次且属实 | 降一级，取消年度晋升资格 |
| **能力退化** | 年度能力评估不达标 | 降一级，强制培训计划 |
| **合规问题** | 泄露客户信息/利益冲突 | 视情节降1-2级或除名 |
| **价值观不符** | 严重违反团队价值观 | 直接除名 |

### 5.2 降级保护机制

- **新手保护期**：晋升后12个月内不触发降级
- **特殊情况豁免**：家庭重大变故/健康问题，经合伙人会议批准可暂停考核
- **降级申诉权**：对降级决定有异议可在15天内申诉

### 5.3 优雅退出机制

#### 主动退出

| 退出类型 | 提前通知期 | 权益处理 | 后续合作 |
|----------|------------|----------|----------|
| 个人发展 | 3个月 | 按实际贡献结算分红 | 保留合作顾问身份 |
| 创业 | 6个月 | 正常结算+知识资产折价回购 | 视情况战略合作 |
| 退休 | 12个月 | 终身荣誉合伙人+顾问费 | 按需返聘 |

#### 被动退出

| 退出类型 | 触发条件 | 处理方式 | 补偿标准 |
|----------|----------|----------|----------|
| 末位淘汰 | 连续4季度绩效末位 | 协商解约 | N+1补偿 |
| 合规除名 | 严重违规/违法 | 立即解除 | 无补偿，保留追诉权 |
| 能力不匹配 | 2次降级后仍不达标 | 协商转岗或解约 | N+1补偿 |

### 5.4 退出交接清单

```
□ 客户项目移交（进行中项目状态+客户联系人）
□ 知识资产移交（方法论文档+工具模板）
□ 团队交接（团队成员分配+待办事项）
□ 财务结算（未结款项+分红结算）
□ 竞业协议签署（如适用）
□ 退出面谈（经验总结+改进建议）
```

### 5.5 校友网络

建立「六六天团校友会」，保持与离职成员的连接：
- 年度校友聚会
- 校友优先合作政策
- 校友推荐客户奖励
- 回聘绿色通道
